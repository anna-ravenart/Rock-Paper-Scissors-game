# Rock Paper Scissors Game in Python

This is a simple console-based Rock Paper Scissors game, player against the computer.

The project was created as a learning exercise and uses only core Python features.

## Gameplay
The computer chooses a random move, and then the player enters their choice.  

The game checks for a winner or a tie each round.  

After each round, the player is asked whether they want to play again.

## Python concepts used
- Tuples
- Loops
- Random

## How to run
Run the Python file in the terminal.
